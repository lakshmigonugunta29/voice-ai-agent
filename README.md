# Real-Time Multilingual Voice AI Agent

## Features
- FastAPI backend
- English, Hindi, Tamil support
- Appointment booking
- Appointment cancellation
- Memory handling
- Slot conflict checking
- OpenAI reasoning
- Latency tracking

## Run Project

pip install -r requirements.txt

uvicorn app.main:app --reload --port 9000
